Domain terms, architecture decisions, and WAC category reference

Glossary PRD Architecture

Domain Terms

TermDefinition
ClientAn adult family home (AFH) using the bot. Each has its own Telegram chat and Google Drive folder. Config-driven in config.json.
DocumentA single file sent to the Telegram channel — photo, PDF, or DOCX. Each processed independently even if sent in a batch.
SandraPrimary human user. The bot's confirmation flow is designed around her phone-based workflow.
New EmployeeA name detected in a document that doesn't match any folder in the client's Drive roster. Triggers automatic folder creation and onboarding flow.

Architecture

DecisionChoice
Bot RuntimeSingle Python script using python-telegram-bot. One systemd service per Telegram bot token (test + production), each sharing the same config.json.
Document ClassificationThree-tier: (1) rules keyword match with word-boundary checks, (2) if ambiguous, the rules result is treated as a hint and elevated to DeepSeek LLM, (3) if LLM fails → manual inline-keyboard correction.
OCR / Text Extractiondoc-extract.py. Uses PyMuPDF + Anthropic Haiku vision for scanned docs. HEIC/HEIF via pillow-heif.
Drive AuthSingle Google service account shared by all clients. Access controlled at the Drive folder level.
Confirmation UXInline keyboard buttons [👍 Yes] [👎 No]. Correction via two sequential text prompts.
Employee RosterScanned from Drive folder names at runtime. Providers flagged in config.json for audit.
Provider vs. CaregiverNo structural folder split. Config marks providers ("providers": ["Sandra Namwase"]). Audit checks 01–08 + facility docs for providers, 01–07 for caregivers.
Multi-TenancySingle bot binary. Client config in config.json keyed by Telegram chat_id.

WAC Categories

Washington Administrative Code chapters 388-76 and 388-112A define 8 document categories for AFH personnel records.

01
Identity & Employment
Photo ID, W-2, I-9, application
02
Background Check
WA name/DOB + national fingerprint
03
Health Screening
TB test within 3 days of hire
04
CPR & First Aid
Valid certs required
05
Orientation & Training
70-hr basic training, HCA, specialty
06
HCA Certification & CE
Cert within 200 days, 12 hrs CE/yr
07
Nurse Delegation
If performing delegated tasks
08
Administrator Training provider
54-hr DSHS-approved admin course

File Naming

{Employee Full Name} - {WAC Category or Description}.{ext}

Example: Fatou Manneh - 04 - CPR & First Aid.pdf
Companion: Fatou Manneh - 04 - CPR & First Aid-robot.txt
